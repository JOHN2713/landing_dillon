# Landing Page - Torneo de Egresados Dillon

Landing page para el registro de jugadores en el Torneo de Egresados del Colegio Dillon - Copa Fausto Rodríguez 2026.

## 🚀 Características

- Diseño responsive optimizado para mobile y desktop
- Formulario de registro de jugadores
- Slider de imágenes automático
- Modal de confirmación de registro exitoso
- Integración con promociones de Perseo y Socio Perseo

## 📋 Estructura del proyecto

```
landing_dillon/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidad JavaScript
└── README.md           # Documentación
```

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (con Flexbox y Grid)
- JavaScript (Vanilla)
- Font Awesome 6.5.1 (para iconos)
- Cloudinary (hosting de imágenes)

## 📱 Responsive Design

El sitio está optimizado para:
- Desktop (1024px+)
- Tablet (768px - 1023px)
- Mobile (hasta 767px)
